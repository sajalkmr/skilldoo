# Problem Statement 1: Skill Swap Platform

# skilldoo

## Team - Sajal, Team 1641

## Setup
```bash
npm install
npx prisma db push
npm run dev
```

## Features
- User authentication & profiles
- Browse and search users by skills
- Send/receive swap requests
- Rating system
- Admin panel

## Tech Stack
- Next.js 15 + TypeScript
- Prisma + SQLite
- Tailwind CSS
- JWT auth
